# DESCRIPCION DEL PROYECTO Y SUS PASOS

- Esta App va a terminar siendo una app donde se puedan 'manejar' un stock, osea un CRUD de productos con algo asi como {id: string, nombre: string, descripcion: string}, La idea es que haya 2 pestañas/ paginas, la primera, con las opciones del CRUD (crear, buscar, actualizar y borrar) -> Productos, y la otra pantalla, una donde esten todos los productos con su descripcion y la cantidad de cada uno, quizas con una busqueda arriba.
- En esta App quiero usar react-navigation, para aprender el tema de las screens, y tambien quiero aprender tailwindcss
- (futura version 1.1)Si la app queda 'ANDANDO' de esa forma BASICA, lo proximo a hacer es poder agregarle fotos a los productos, y poder agregar los productos escaneando el codigo de barras con la camara del celular. Para practicar el tema de usar cosas del celular como la camara
- (futura version 1.2)Lo proximo a hacer es que la App envie notificaciones cuando un producto baje de un stock minimo, de esta forma se sabe cual hay que salir a reponer primero.
